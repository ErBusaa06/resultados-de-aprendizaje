# Resultados de Aprendizaje (RA) – Formación Profesional

Proyecto web orientado a la visualización y organización de Resultados de Aprendizaje (RA) para Formación Profesional.  
Incluye páginas HTML, transformaciones XSL/XSLT, archivos JSON y otros recursos necesarios para mostrar la información de forma estructurada.

## Estructura del proyecto
El proyecto se organiza dentro de la carpeta principal:

- **public_html/**
  - **pagina web/**
    - formacion_profesional.html (página principal)
    - Archivos HTML adicionales
    - Archivos XSL y XSLT
    - Archivos JSON
    - Recursos multimedia
- **imagenes/** (capturas de ejemplo para el README)

## Funcionalidad
- Visualización de Resultados de Aprendizaje (RA)
- Transformación de datos mediante XSL/XSLT
- Carga de información desde archivos JSON
- Navegación entre diferentes apartados del proyecto
- Estructura modular y organizada

## Cómo usar el proyecto
1. Clona el repositorio o descárgalo.
2. Abre el archivo **formacion_profesional.html** desde la carpeta `public_html/pagina web/`.
3. Asegúrate de mantener la estructura de carpetas para que las rutas funcionen correctamente.
